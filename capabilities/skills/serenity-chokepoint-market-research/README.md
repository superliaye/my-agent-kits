# Serenity-Derived Chokepoint Market Research — Artifact Index

A source-grounded distillation of the market-research *methodology* of Serenity (`@aleabitoreddit`). Research framework only — **not** financial advice, not a trading signal, not a personality profile.

## Files

| File | What it is |
|---|---|
| [`SKILL.md`](SKILL.md) | The reusable, standalone skill. Start here to *apply* the method. |
| [`templates/theme_analysis.md`](templates/theme_analysis.md) | Portable 20-section template for analyzing any theme. |
| [`examples/example_power_transformers.md`](examples/example_power_transformers.md) | Worked example: framework applied to large power transformers using current 2025-26 primary sources. |
| `_unshipped/serenity_methodology_report.md` | Full evidence report: 13 principle cards, each with citations, classification, failure modes, tensions; plus track-record analysis, QC self-audit, and limitations. |
| `_unshipped/serenity_corpus_manifest.csv` | 785 methodology-relevant posts with canonical URLs, excerpts, tags, attribution/retrieval status, and a coverage header. |
| `_unshipped/serenity_full_retrieved_corpus.csv` | The complete retrieved archive (5,857 posts, 2025-07-02 to 2026-06-08) for reproducibility. |

> The `_unshipped/` files are maintainer-only provenance/reproducibility assets. They stay in the kit repo but are excluded from deployed installs of this skill (see the kit's `_unshipped/` convention), so a consumer who installs the skill gets only `SKILL.md`, `README.md`, `templates/`, and `examples/`.

## Provenance & honesty (summary)
- **Primary evidence:** Serenity's public X posts, obtained via the third-party archive `github.com/yan-labs/serenity-aleabitoreddit` (preserves canonical X URLs + timestamps). Every principle cites `x.com/aleabitoreddit/status/...`.
- **Hard limitation:** direct X.com fetch is paywalled (HTTP 402); IDs were verified against the archive, not re-fetched from X. Default attribution is therefore `probable`, not `verified`.
- **Coverage:** *all publicly accessible posts retrieved during this run* — **not** all posts ever published. Pre-2025-07 and deleted/private posts are not captured. Self-reported returns are unaudited.

See the report's "Source-of-Truth Policy & Coverage" and "Limitations" sections for the full disclosure.
